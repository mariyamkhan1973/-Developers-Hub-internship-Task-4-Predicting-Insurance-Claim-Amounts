# Task 4: Predicting Insurance Claim Amounts

### What I Did

In this task, I worked on predicting insurance claim amounts using customer information such as age, BMI, and smoking status. I first explored the dataset and then created visualizations to understand how different factors affect insurance charges.

After analyzing the data, I trained a Linear Regression model to estimate insurance claim amounts and evaluated its performance using standard regression metrics.

### My Observations

- Customers with higher BMI values generally tend to have higher insurance charges.
- Smoking has a significant impact on insurance claim amounts, as smokers usually have much higher charges than non-smokers.
- Age also plays an important role, with older customers often having higher insurance costs.
- The visualizations helped identify the relationship between personal factors and insurance charges.
- The Linear Regression model was able to capture the overall trend in the data and make reasonable predictions.


### Conclusion

This task helped me understand the basics of regression modeling and how machine learning can be used to predict numerical values. I also learned how to visualize data, analyze feature relationships, and evaluate model performance using regression metrics.
